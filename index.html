Cajita abierta

platform for sensor interfaces

WHAT IS IT?

"Cajita Abierta" is breakout wireless sensor circuit ESP32-based sensor and actuator board for capturing live sensor data and playback, controlling lights and motors.

Sensors implemented so far : Air pressure (breath pressure), Accelerometer (XYZ angle and acceleration on the three axis), breathing sensors, Analog sensors inputs such as light and Carbon Monoxide (extendable).

It works through a mobile application, where you select where to send the data: Hydra, Chataigne, ossia, touchOSC server and the Technologies de la fÃªte server module. Is using UDP, OSC, MIDI protocols (mdns discovery for negotiating UDP sending port)

COMMUNITY

For tutorial of how to use it

We offered a workshop to the community at Gimnasio de Arte in CDMX, Mexico

Interconnections, circuits and sensors: small data for the creation of images and sounds. As opposed to Big Data, we will discover how sensors allow us to record the small, unnoticed events around us, such as subtle gestures, heartbeats, breaths and the like. We will then explore how this small data can be harnessed to create sounds, images, performances or installations.

PINCH TO AWAKEN VR

Voltage divider with variable resistance

A simple electrical circuit that divides the voltage of a source between one or more resistors connected in series and in which its voltage drop is directly proportional to its ohmic value. A fixed resistor and a variable resistor are used, in this case a conductive wool fabric, which when stretched changes the resistance value of the circuit. The voltage divider equation assumes that you know three values from the previous circuit: the input voltage (Vin) and both resistance values (R1 and R2). Given those values, we can use this equation to find the output voltage (Vout): Vsal = Vin * (R2 / (R1 + R2) )

INTERACTIVE GLOVE 'K4'

wireless MIDI/CC controller for music and live visuals control.

This project was made in the framework of the presentation of the third show of the cycle "I don't know who I am" of the artist K4, where he proposed to deepen in each of the 3 characters that make up the album of the same name, creating for each date a theme and a production where art, stage design, costumes, accompanied the development of technologies that allow the maximum verbal / non-verbal expression of each of his characters, the latter has a science fiction aesthetic, with a technological development close to post-humanism.
